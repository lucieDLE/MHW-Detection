# MHW-Detection

This project builds an interactive dashboard to explore Sea Surface Temperature (SST) anomalies over the Mediterranean Sea using xarray and Panel/hvPlot. The application displays a spatial anomaly map where users can click on any location to dynamically generate associated time-series visualizations, including raw anomalies, rolling averages, and distribution histograms.
The system is designed for climate data exploration, enabling intuitive spatial-to-temporal analysis through linked interactive plots. The architecture emphasizes clean data handling with xarray, reactive visualization using Holoviews streams, and a structured Panel layout for a professional dashboard interface
